# claude-marketplace

Personal Claude Code skills.

## Install

```bash
claude plugin install Leolebleis/claude-marketplace
```

## Skills

- **french-writing** -- Auto-invoked when writing French text. Accent rules, contractions, gaming terms, tone guidelines.
- **google-tasks** -- Manage Google Tasks. List, create, complete, update, search tasks.
- **pc-performance-audit** -- Audit a remote Windows PC via SSH for performance bottlenecks. Diagnoses CPU, RAM, GPU, storage, bloatware, and produces a ranked fix plan with apply/verify/rollback commands.
- **game-tuning** -- Per-game optimization on Windows: G-Sync 101 stack, DLSS/Frame Gen, Reflex, per-app NVCP profiles, launch options, in-game video settings. Includes a CS2-specific cookbook (file paths, convars, launch options, in-game settings, pro comparison).
- **solved-problem** -- Before building custom, check if someone already solved it. Dispatches parallel research subagents across GitHub, Reddit, HN, awesome-lists, and blogs; returns ranked tools with pros/cons, learnings, and a recommendation.
